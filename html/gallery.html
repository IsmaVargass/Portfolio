Plataforma Web de Sostenibilidad

Plataforma web para Iberdrola enfocada en la sostenibilidad y alineada con los objetivos de la Agenda 2030. Esta plataforma permite la compra y venta de productos de segunda mano, similar a Vinted, fomentando así la economía circular y el consumo responsable.

Para su desarrollo, utilicé HTML, CSS y Bootstrap para crear una interfaz atractiva, responsiva y fácil de navegar. Con Bootstrap, aseguré un diseño moderno y adaptable a cualquier dispositivo. Además, implementé JavaScript para mejorar la interactividad y la experiencia del usuario, permitiendo funciones dinámicas como filtros, animaciones y actualizaciones en tiempo real.

El objetivo de esta web es reducir el impacto ambiental al promover la reutilización de productos, ayudando a minimizar residuos y a optimizar recursos. Con esta iniciativa, Iberdrola refuerza su compromiso con la sostenibilidad, ofreciendo a los usuarios una alternativa eficiente y ecológica para dar una segunda vida a los objetos.

Juego en Java 2D - Astralis

Desarrollé un juego en Java 2D con Pixel Art, inspirado en los juegos retro de los 90. Utilicé Java con Swing/JavaFX, implementando un motor gráfico optimizado para sprites, animaciones y colisiones, además de un sistema de física y efectos visuales mejorados.

El diseño visual incluye Pixel Art detallado y una paleta vibrante, garantizando una experiencia fluida e inmersiva. Este proyecto combina innovación y nostalgia, rindiendo homenaje a los videojuegos clásicos.

BookTrack - Gestión de Libros

BookTrack es una plataforma colaborativa para la gestión de libros, desarrollada en GitHub utilizando GIT para el control de versiones. Mediante ramas, pull requests y Git Flow, garantizamos un desarrollo organizado y sin conflictos.

La plataforma permite añadir, clasificar y buscar libros fácilmente, con una arquitectura modular que facilita su escalabilidad. Este proyecto demuestra el potencial del trabajo en equipo con GIT y GitHub, promoviendo buenas prácticas en el desarrollo de software.
